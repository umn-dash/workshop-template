Install Zoom

If you haven't used Zoom before, go to the official website to download and install the Zoom client for your computer.

In this workshop, you will be "coding along" with the Instructors. We recommend using either two monitors, two machines, or dividing your screen so that you can both watch the zoom presentation and code at the same time. This blog post includes detailed information on how to set up your screen to follow along during the workshop.
